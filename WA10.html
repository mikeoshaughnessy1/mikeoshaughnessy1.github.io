Why did i chose to add slide numbers?

  • Slide numbers are needed so you can tell where you are in the slide deck.
  • The other reveal.js animations were really hard to implement.

To display slide numbers, all you have to do is set the slideNumber configuration option to true!


					Reveal.initialize({
						slideNumber: true
						});

Note: This code will need to be added to the bottom of your file.
